This is a web application developed where user can upload the image of their dog's affected skin into the system. The fine-tuned model loaded in the
application make the inference and return the detection result. The detection results consists of the skin disease name and confidence level is 
displayed to the user. Google colab is the platform used to fine tuning the model. The model is fine-tuned and evaluated using various performance metrics such as the precision, recall, F1 Score and confusion 
matrix. Once the model achieves the good validation accuracy, it will be loaded into the application.
